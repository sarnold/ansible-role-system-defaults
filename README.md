Ansible Role for managing default stuff
=======================================

My default apps and configuration stuff I do not want to miss.

Supported Distributions
-----------------------

- Gentoo
- CentOS 7

Requirements
------------

None.

Role Variables
--------------

None.

Dependencies
------------

None.

Example Playbook
----------------
```
- hosts: all
  roles:
    - role: system-defaults
```
